# Demo Script

Hi, this is Solana Signal Scout.

It is an autonomous Solana research agent built for the OOBE / Ace Data Cloud autonomous agent bounty.

The agent performs a complete flow:
1. It loads its configuration.
2. It registers a Synapse/OOBE-style autonomous agent identity.
3. It requests a protected Solana wallet intelligence resource.
4. It completes an x402-style payment step.
5. It runs Ace Data Cloud-style analysis.
6. It generates a final wallet intelligence report.

The report includes a target wallet, payment status, risk score, opportunity score, signals, and a final recommendation.

The project includes both a command-line demo and a local web dashboard.
